## 1. 目的・背景

スマホネイティブ世代（20代）を主なターゲットとし、対面での保険加入に抵抗のあるユーザー層に対して、**スマホだけで保険の見積もり・申込が完結できるアプリ**を提供することで、販路を拡大する。

---

## 2. スコープ（MVPで実現する機能）

### ✅ MVPで実装する機能

| 分類         | 機能名                     | 概要                           |
|--------------|----------------------------|--------------------------------|
| 認証・認可   | ユーザ登録／ログイン／ログアウト | Keycloak連携                    |
| 商品情報     | 保険商品一覧表示           | 商品名・概要の表示              |
| 見積もり     | 保険商品の見積もり取得     | 入力条件に応じた結果を表示      |
| 見積管理     | 見積もり履歴確認           | ユーザ単位で履歴を表示          |
| 申込         | 保険申込処理               | 入力内容を元に申込情報を作成    |
| 契約情報     | 契約一覧表示               | 申込済みの契約情報を表示        |

### ⛔ 将来実装予定

- 電子マネー決済機能（PayPay, LINE Pay 等）
- AIによるプラン提案、見積補助
- 契約の更新／解約手続き
- 管理者画面（保険会社・代理店向け）

## 3. バリュープロポジション（提供価値）

- スマホ完結型で若年層にフィット
- 手間なく保険比較・加入できる体験
- 従来の営業販売に代わるデジタル販路

## 6. 非機能要件

- モバイルファーストなUI/UX
- セキュリティ：最低限のOAuth2対応
- シンプルなCI/CDで自己完結できる構成

## 8. リスクと制約

| リスク内容                      | 対応策                     |
|-------------------------------|----------------------------|
| 開発スピードの低下            | 優先度を明確に段階的開発   |
| Keycloak等の学習負荷          | 最低限の連携からスタート   |

## 10. 販売する保険商品

### 🌟 個人年金保険

#### 【特徴】

| 特徴           | 内容                                                             |
|----------------|------------------------------------------------------------------|
| 保険料         | 定額払い・払い込み満了後に返戻金あり                             |
| 予定利率       | 契約時／15年後／10年ごとに変動、最低保証0.5%                     |
| 告知           | 健康情報の告知は不要                                             |
| 死亡保障       | 払込済額を返金                                                   |
| 受取形式       | 一括または年金、代理受取可                                       |

#### 【見積もりに必要な条件】

| 条件項目       | 内容                                             |
|----------------|--------------------------------------------------|
| 生年月日       | 必須                                             |
| 性別           | 男・女                                           |
| 月額保険料     | ¥10,000～¥50,000（¥1,000単位）                   |
| 払込期間       | 15年以上（MAX 65歳まで）                        |
| 税制適格特約   | あり／なし選択式                                 |

#### 【見積もり結果（各シナリオ）に必要な情報】

| 項目                   | 内容例                                                      |
|------------------------|-------------------------------------------------------------|
| 予定契約日             | 毎月1日開始（例：6月申込→7月1日契約）                     |
| 契約時の予定利率       | 動的取得                                                    |
| 払込保険料総額         | 自動計算                                                    |
| 保険料払込期間         | 入力値に基づく                                              |
| 年金開始年齢           | 自動算出 or 指定                                           |
| 所得控除額             | 税制適格特約に応じて決定                                   |
| シナリオ別返戻額       | 想定利率ごとの累計額・年金額・一括額・返戻率               |

#### 【申し込みに必要な情報】

- 上記「見積もりに必要な条件」に加えて：
  - 年金の受け取り方法（一括／年金）
  - 年金開始年齢または一括受取年齢
  - 受取回数（年金／一括で分岐）
  - 代理受取人情報（複数可）

#### 【運用条件（架空）】

| 資産分類 | 割合 |
|----------|------|
| 国債     | 50%  |
| 社債     | 30%  |
| 株式     | 20%  |

### 🎓 学資保険

#### 【特徴】

| 特徴           | 内容                                                         |
|----------------|--------------------------------------------------------------|
| 用途           | 教育資金準備                                                 |
| 払込方法       | 月払・年払・全期前納                                         |
| 祝金受取       | 18〜21歳時に4回分割                                          |
| 死亡時保障     | 払込免除・保障継続                                           |
| 税制優遇       | 生命保険料控除対象                                           |

#### 【見積もりに必要な条件】

| 項目                   | 内容                                 |
|------------------------|--------------------------------------|
| 子どもの生年月日       | 出生〜2歳未満限定                   |
| 契約者の情報           | 氏名、生年月日、性別               |
| 保険料額               | 月額：1〜2万円、年額：12〜24万円   |
| 払込方法               | 月払・年払・全期前納               |

#### 【見積もり結果に含める情報】

| 項目                   | 内容                                 |
|------------------------|--------------------------------------|
| 契約日                 | 毎月1日開始                         |
| 払込総額・期間・方法   | 入力に応じて決定                   |
| 控除対象額             | 年額計算（税制参考）               |
| 総受取額               | 祝金合計、年別金額                 |
| 返戻率                 | 自動算出（%）                       |
| 解約返戻金             | 年数別シミュレーション             |

#### 【申し込みに必要な情報】

- 子どもの生年月日
- 契約者情報（氏名・生年月日・性別）
- 保険料・払込方法
- 予定契約日（毎月1日）
- 祝金受取人（通常は子ども）
- 振込先予定口座情報

#### 【運用条件（架空）】

| 資産分類     | 割合  |
|--------------|-------|
| 国債         | 60%   |
| 社債         | 20%   |
| 安全資産     | 20%   |
| 利率         | 年1.8% 固定 |
| 解約返戻水準 | <5年: 70〜80%、5年: 元本保証、満了後: 130% |
